Senior BackEnd Engineer

Windward is a maritime data analytics company shedding light on what’s really happening on the seas. Backed by some of the world’s leading funds and private individuals in the fields of intelligence and startups , we aim to be the True North for everything maritime. It’s an exciting and tough journey, and we are looking for a Senior Java Backend Engineer to join our team.

As a Senior BackEnd Engineer at Windward you will be using cutting-edge technologies, such as Apache Spark and Cassandra, to build a scalable distributed architecture for handling maritime big data. You will be part of a small, tight-knit, dynamic, hands-on team that takes pride in its exceptional coding standards and development practices. As part of our team, you’ll be dealing with massive amounts of data from different sources and using machine learning algorithms to identify behavioral patterns in the data. Each member of our team is a development athlete/all-around-player. You should be able to build a product end-to-end from design to coding, testing and deployment.

We”re located in Tel Aviv, on the 35th floor in Edgar 360 building, enjoying the amazing ocean view!

  • 3-5 years’ experience in server side development (Java / Scala / C# )
  • Experience with NoSQL data stores like Apache Cassandra, MongoDB, and Redis
  • Good understanding of web applications (JSON, REST API)
  • Experience with EC2, S3, Machine Learning – advantage
  • Experience with big data programming, preferably using Apache Spark – advantage
  • Strong communications and problem-solving skills
  • A passion for quality and bringing value to our users
  • Intelligent and self-motivated
